## 概要
  政策論題、価値論題、事実論題など2択の問いかけを設定し、その主題について賛成派、反対派の異なる立場に別れチャット形式で討論することができるアプリケーションです
  
## 使い方の例
  * ニュース、時事の賛成、反対を聞いてみる
  * みんなはどっち派？を聞いてみる
  * 迷っていることを相談してみる 
  * 気になる商品を比較してみる 
  * 旅行先の候補を比較してみる 
  * 音楽や漫画など好きなことで分かれる意見を語り合う 
  * 自分のコメント力を試してみる
  
## 制作背景
  * LINEやSlackなどのチャットアプリで決議をとりたい時、賛成派、反対派にわかる機能があればいいなと思いました。
